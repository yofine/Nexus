# Keyboard Shortcuts

## Global shortcuts

| Shortcut | Action |
| --- | --- |
| `Cmd/Ctrl + K` | Open the command palette |
| `Cmd/Ctrl + N` | Create a new agent pane |
| `Cmd/Ctrl + W` | Close the active pane |
| `Cmd/Ctrl + 1-9` | Switch panes by index |
| `Cmd/Ctrl + G` | Open Git diff / Review |
| `Cmd/Ctrl + ,` | Open Settings |
| `Cmd/Ctrl + \`` | Toggle the bottom terminal |

## Dialog shortcuts

| Shortcut | Action |
| --- | --- |
| `Escape` | Close the current dialog or command palette |

## Usage tips

- `Cmd/Ctrl + K` and `Cmd/Ctrl + N` cover most high-frequency actions
- `Cmd/Ctrl + G` is the fastest route to inspect diffs
- number shortcuts are faster than the mouse when hopping between panes
