# Nexus Docs

Nexus is a local web console for managing multiple CLI AI agents in parallel from one browser interface.

## What you can do with it

- Run multiple agent panes in the same workspace
- Inspect pane state, session metadata, file changes, and Git diffs
- Isolate parallel tasks with Git worktrees
- Replay previous sessions and review notes and file activity
- Adjust themes, fonts, and agent definitions from the settings UI

::: info Note
This site is user-facing documentation. It follows the current repository behavior, and it includes experimental capabilities that are already visible in the product.
:::
