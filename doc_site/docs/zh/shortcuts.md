# 快捷键

## 全局快捷键

| 快捷键 | 作用 |
| --- | --- |
| `Cmd/Ctrl + K` | 打开命令面板 |
| `Cmd/Ctrl + N` | 新建 Agent Pane |
| `Cmd/Ctrl + W` | 关闭当前活动 Pane |
| `Cmd/Ctrl + 1-9` | 按序号切换 Pane |
| `Cmd/Ctrl + G` | 打开 Git Diff / Review |
| `Cmd/Ctrl + ,` | 打开 Settings |
| `Cmd/Ctrl + \`` | 切换底部终端 |

## 对话框快捷键

| 快捷键 | 作用 |
| --- | --- |
| `Escape` | 关闭当前对话框或命令面板 |

## 使用建议

- 高频操作优先用 `Cmd/Ctrl + K` 和 `Cmd/Ctrl + N`
- 需要快速观察差异时直接用 `Cmd/Ctrl + G`
- 频繁切换 pane 时，用数字键比鼠标更快
