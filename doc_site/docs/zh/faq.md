# FAQ / 排障

## 为什么启动后没有看到某个 agent 类型？

如果该 agent 在本机未安装，新增 pane 时它会显示为不可用。先完成对应 CLI 的安装，再重新打开对话框。

## 为什么恢复会话列表是空的？

可能原因包括：

- 当前 agent 类型没有历史 session
- session 不在当前可发现路径下
- 该 agent 的历史记录不兼容当前恢复流程

## 为什么文件树或活动没有更新？

Nexus 依赖文件监听和终端活动解析。短时间内没有变化、被忽略的路径，或某些操作没有产生可识别输出时，界面更新会不明显。

## 为什么 worktree pane 无法恢复？

恢复需要原来的分支和 worktree 路径仍然有效。如果对应分支已经被删除，系统可能跳过该 pane。

## 为什么 Replay 里没有内容？

Replay 只记录可回放的 agent 会话。全局 shell 不会像普通 agent pane 那样进入回放体系。

## `mexus stop` 没有效果怎么办？

先确认当前端口是否真有 Nexus 在运行，再检查你是否在同一台机器、同一端口环境下执行命令。

## 文档里为什么会出现 Experimental 标记？

因为这个站点覆盖“仓库里当前已经存在的用户功能”，其中有些能力已经可见，但仍可能继续调整交互、命名或入口位置。
