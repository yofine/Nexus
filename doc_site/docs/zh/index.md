# Nexus 文档

Nexus 是一个本地 Web 控制台，用来在单个浏览器界面里并行管理多个 CLI AI Agent。

## 你可以用它做什么

- 在一个工作空间里同时运行多个 agent pane
- 查看 pane 状态、会话元信息、文件变化和 Git diff
- 用 worktree 隔离并行任务，减少分支互相干扰
- 回放历史会话，查看 Notes 和文件活动轨迹
- 通过设置页调整主题、字体和 agent 定义

::: info 说明
本文档面向最终用户，内容以当前代码实现为准。已经出现在产品中的实验性能力也会收录，但会明确标注状态。
:::
